🚀 The Cloud Engineer's Git Cheat Sheet

A categorized reference for my daily version control workflows.

Core Workflow

  • git init Create new local repo
  • git status Check changes & staging
  • git add . Stage all changes
  • git commit -m "msg" Save snapshot
  • git log --oneline View short history
  • git checkout [hash] Recover previous state

Branching

  • git branch List all branches
  • git switch Switch to branch
  • git switch -c Create + Switch
  • git merge Merge into current
  • git branch -d Delete merged branch
  • git branch -D Force delete branch

Remote & GitHub

  • git remote add origin [url] Link to GitHub
  • git push -u origin main First push (upstream)
  • git push Upload commits
  • git pull Download updates
  • git clone [url] Download entire repo
  • git remote -v List remote URLs

Authentication (SSH)

  • ssh-keygen -t ed25519 Generate SSH Key
  • eval "$(ssh-agent -s)" Start SSH agent
  • ssh-add ~/.ssh/id_ed25519 Add key to agent
  • cat ~/.ssh/id_ed25519.pub Copy Public Key
